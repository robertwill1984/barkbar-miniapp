## 不可违反的财务原则

以下概念必须分别记录、分别统计：

- 实际收款；
- 经营收入；
- 尚未兑付会员权益；
- 退款；
- 运营支出；
- 现金余额。

所有汇总必须支持下钻到订单、支付、核销、收入确认、退款、支出或现金流水。财务、权益、库存和操作日志不得物理删除。

后端还必须支持股东视角财务分析：

- 现金流分析；
- 内部经营利润测算；
- 股东分红依据。

系统只提供内部经营口径和分红测算依据，不自动把内部经营利润等同于法定会计利润、税务利润或可直接分红金额。正式分红前需要会计/税务/法律复核。
