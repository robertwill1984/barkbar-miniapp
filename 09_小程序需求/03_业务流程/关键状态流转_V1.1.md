# 关键状态流转 V1.1

日期：2026-07-09

## 1. 预约与权益锁定

```mermaid
stateDiagram-v2
    [*] --> DRAFT
    DRAFT --> PENDING_PAYMENT: 散客提交
    DRAFT --> BENEFIT_LOCKED: 次卡锁1次
    DRAFT --> PENDING_REVIEW: 需人工审核
    PENDING_PAYMENT --> PENDING_REVIEW: 支付成功且需审核
    PENDING_PAYMENT --> CONFIRMED: 支付成功且自动通过
    BENEFIT_LOCKED --> PENDING_REVIEW
    BENEFIT_LOCKED --> CONFIRMED
    PENDING_REVIEW --> CONFIRMED: 审核通过
    PENDING_REVIEW --> REJECTED: 审核拒绝/解锁或退款
    CONFIRMED --> CHECKED_IN
    CHECKED_IN --> COMPLETED
    CONFIRMED --> CANCELLED
    CONFIRMED --> NO_SHOW: 超过60分钟
    CONFIRMED --> STORE_CLOSED_CANCELLED
```

取消或拒绝必须解锁次卡；散客按规则退款/改期。

## 2. 动态码与核销

```mermaid
stateDiagram-v2
    [*] --> ISSUED
    ISSUED --> CONSUMED: 60秒内首次合法核销
    ISSUED --> EXPIRED: 超过60秒
    ISSUED --> REVOKED: 卡冻结/用户刷新
    CONSUMED --> [*]
    EXPIRED --> [*]
    REVOKED --> [*]
```

同一nonce、幂等键或核销业务号不可重复成功。

## 3. 弱网补录

```mermaid
stateDiagram-v2
    [*] --> PENDING
    PENDING --> VALIDATING: 网络恢复/店长提交
    VALIDATING --> POSTED: 无重复且资格有效
    VALIDATING --> DUPLICATE_REJECTED: 已存在核销
    VALIDATING --> OWNER_REVIEW: 超限/跨日/资格异常
    OWNER_REVIEW --> POSTED: 老板批准
    OWNER_REVIEW --> CLOSED: 拒绝
```

## 4. 卡项暂停与到期

```mermaid
stateDiagram-v2
    [*] --> PENDING_ACTIVATION
    PENDING_ACTIVATION --> ACTIVE
    ACTIVE --> PAUSE_REQUESTED
    PAUSE_REQUESTED --> PAUSED: 批准
    PAUSE_REQUESTED --> ACTIVE: 驳回
    PAUSED --> ACTIVE: 恢复并顺延
    ACTIVE --> EXPIRED_PENDING_ACTION: 到期
    EXPIRED_PENDING_ACTION --> CLOSED: 权益结项
    EXPIRED_PENDING_ACTION --> ACTIVE: 延期
    EXPIRED_PENDING_ACTION --> REFUNDING
    EXPIRED_PENDING_ACTION --> DISPUTED
    EXPIRED_PENDING_ACTION --> EXCEPTION_TODO: 30日未处理
    REFUNDING --> REFUNDED
```

暂停期间停止有效期和逐日内部收入摊销。

## 5. 退款

```mermaid
stateDiagram-v2
    [*] --> REQUESTED
    REQUESTED --> MANAGER_APPROVED
    REQUESTED --> REJECTED
    MANAGER_APPROVED --> FINAL_APPROVED
    MANAGER_APPROVED --> REJECTED
    FINAL_APPROVED --> CHANNEL_PROCESSING
    CHANNEL_PROCESSING --> SUCCEEDED
    CHANNEL_PROCESSING --> FAILED
    FAILED --> CHANNEL_PROCESSING: 重试
    SUCCEEDED --> REVERSED: 退款冲正
```

普通员工申请，店长初审，老板/授权财务终审；禁止自批。


## 7. 协议

```mermaid
stateDiagram-v2
    [*] --> DRAFT
    DRAFT --> REVIEWED
    REVIEWED --> PUBLISHED
    PUBLISHED --> RETIRED: 新版本发布
    PUBLISHED --> RECONFIRM_REQUIRED: 重大变化
    RECONFIRM_REQUIRED --> ACCEPTED: 用户签署
```

旧版本和签署证据永久保留在规定保存期内，不覆盖。
