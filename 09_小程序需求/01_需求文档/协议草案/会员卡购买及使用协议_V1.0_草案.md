# 会员卡购买及使用协议 V1.0（草案）

> 上线前法律和会计复核。购买页须以显著方式展示价格、期限、退款、暂停和限制。

## 1. 卡项

10次卡、季卡、年卡及其他卡项，以购买时规则快照为准。不得在售出后单方面新增不利规则。

## 2. 使用

- 10次卡本人到场、每日最多3次；标准和赠送次数均为实际可用次数。
- 季/年卡最多2名验证使用人，共享权益，每日限用1次。
- 动态码有效60秒、截图无效；会员权益不替代入园核验。
- 超基础人数/犬数按当时价格或页面提供的次卡扣次选择处理。

## 3. 权益

季/年卡每自然日最多1杯指定赠饮；指定首购卡可领礼品包。零元权益不折现，领取和成本留痕。

## 4. 暂停、到期与退款

适用《会员卡暂停延期与退款规则》。到期进入30日处理期，不自动清零合法权利。退款逐项展示计算并经分级审批。

## 5. 家庭成员

第二使用人由主会员邀请验证；有效期内免费更换一次，再次更换须批准；解绑后码失效并保留历史。

## 6. 异常

倒卖、代核销、冒用、虚假邀请等可依法和依约冻结调查，但不得影响用户依法享有的主要权利。
