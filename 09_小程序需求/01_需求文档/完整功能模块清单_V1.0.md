# Bark & Bar 微信小程序｜完整功能模块清单 V1.0

版本：V1.0｜日期：2026-07-09

## 1. 顾客微信小程序

### 1.1 账号与档案

- 微信授权登录、手机号绑定、注销申请。
- 昵称、头像、常用联系人。
- 宠物：姓名、品种、性别、生日/年龄、体重、照片、犬证、疫苗、驱虫、绝育、健康、行为风险和备注。
- 主人与宠物关系、默认出行宠物。
- 用户协议、隐私政策、入园协议的版本、确认时间、IP/设备证据。

### 1.2 预约入园

- 日期、时段、人数、犬数、宠物、票/卡选择。
- 展示剩余容量、开放/闭园状态和最晚预约/取消时间。
- 协议确认、健康声明、预约审核。
- 待支付、待审核、已确认、已签到、已完成、已取消、爽约状态。
- 修改、取消、历史记录、到期提醒。

### 1.3 门票与会员卡

- 单次票、10次卡、季卡、年卡、后台配置的新卡。
- 价格、促销、有效期、次数、适用人数/犬数、绑定人、退款、暂停、延期和赠送权益。
- 购买、开卡、绑定家庭成员、剩余次数/天数、使用明细。
- 动态二维码/短码、防截图、冻结/过期提示。

### 1.4 商城与点单

- 分类、商品、规格、加料、会员价、活动价、库存/售罄。
- 购物车、微信支付、店内取餐/到店自取、订单进度。
- 退款、售后、电子凭证。

### 1.5 会员与营销

- 优惠券、积分（首期可关闭）、老带新邀请码/海报。
- 活动报名、名额、规则、核销和奖励。
- 到期、续费、沉睡召回和生日权益。

## 2. 门店管理后台

### 2.1 工作台

- 今日预约、预计人数/犬数、待审核、待签到、待核销、异常提示。
- 快速扫码、手工短码、预约核销、点单和事故入口。

### 2.2 会员

- 用户/宠物档案、风险标签、黑名单。
- 卡项、次数、有效期、核销、到店频率、消费和售后。
- 冻结、恢复、延期、转卡、次数调整、退款；均需原因、权限和审计。

### 2.3 预约与入园

- 日历/时段列表、容量、闭园、审核、签到、完成、取消、爽约。
- 入园六问、证件健康核验、绿/黄/红管控、禁入原因。
- 当日人员/犬只数量、泳池需求和异常记录。

### 2.4 卡项

- 卡模板、规则版本、复制、上下架、促销、权益包。
- 销量、收款、确认收入、待兑付、到期和退款。
- 不允许修改已售卡的锁定财务政策。

### 2.5 商品订单与库存

- 商品、规格、成本、售价、会员价、上下架。
- 订单、支付、制作/备货、交付、退款、售后。
- 采购、入库、销售/领用/报损出库、退货、盘点、临期和供应商。

### 2.6 活动

- 会员专属、优惠券、满减、购卡赠送、生日、续费、召回、老带新。
- 人群、预算、名额、叠加规则、触达、参与、核销、增量收入和 ROI。

### 2.7 财务

- 支付对账、退款、实际收款、收入确认、待兑付、现金流水。
- 房租、人工、水电、采购、营销、设备、维修、耗材、税费、平台费等支出。
- 财务调整、到期待确认、月结锁定、导出。

### 2.8 系统

- 员工、角色、权限、数据范围、审批流。
- 规则参数、消息模板、字典、文件、日志、备份和任务监控。

## 3. 老板经营数据中心

- 今日/月实际收款、确认收入、退款、支出、净现金流和现金余额。
- 次卡未核销余额、季卡/年卡待兑付金额、到期待确认金额。
- 有效/新增/到期会员、到店频率、续费率、沉睡/高频会员。
- 预约转化、到店率、爽约率、客单价和二次消费率。
- 商品销量、毛利、库存周转、积压、临期、缺货。
- 现金覆盖月数、收入趋势、支出异常、会员兑付压力和未来现金预测。
- 所有指标均可下钻至明细与公式。

## 4. 横向基础能力

- 微信登录、微信支付、退款、订阅消息。
- 幂等键、分布式锁/数据库锁、防重和重试。
- 文件存储、图片压缩、内容安全。
- 权限、字段脱敏、加密、操作日志。
- 测试/生产隔离、配置中心、监控告警、备份恢复。
- 数据导入导出、时间和金额统一规则。

## 5. 首期优先级

- P0：登录/权限、档案、协议、预约、票卡、支付、核销、基础财务六账、支出、日志。
- P1：商城订单、库存、完整报表、退款审批、老带新。
- P2：活动、优惠券、智能经营提醒、复杂积分和预测。
